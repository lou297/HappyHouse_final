## 목표
- 웹 아키텍처를 이해하고 활용할 수 있다.
- Vue.js를 활용하여 SPA 웹 서버를 구축할 수 있다.
- MVC 아키텍처를 활용하여 확장성과 유지보수성 좋은 웹 서버를 구축할 수 있다. 

## 구현 기본(필수) 기능
1) 메뉴에 QnA 게시판 추가</br>
2) 게시글 목록 서비스</br>
3) 게시글 입력 서비스</br>
4) 게시글 수정, 삭제, 상세 검색 서비스</br>
